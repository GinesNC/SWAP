#Ejercicio T2.2:
***Buscar frameworks y librerías para diferentes lenguajes que permitan hacer aplicaciones altamente disponibles con relativa facilidad.***
***Como ejemplo, examina PM2***
***https://github.com/Unitech/pm2***
***que sirve para administrar clústeres de NodeJS.***

PM2 soporta los principales frameworks y cualquier aplicacion de nodeJS.
Algunos de los framworks son:
##[Sails] (http://sailsjs.org/)
Segun en su pagina web, este frameworks es especialmente bueno para chats o juegos multijugador, pero se puede usar en cualquier aplicacion web.
Algunas de sus caracteristicas son:
 - Es 100% compatible con javascript.
 - Funciona con cualquier base de datos.
 - Tiene websocket de apoyo sin codigo adicional.
 - ... 

##[experss] (http://expressjs.com/)
En su pagina web dicen que es una infraestructura web para aplicaciones en Node.js, que es facil y rapido crear API y que no oculta las caracteristicas del servidor web Node.js.

##[hapi] (http://hapijs.com/)
es un frameworks para aplicaciones y servicios de construcción. 
HAPI permite a los desarrolladores centrarse en realizar aplicaciones reutilizables en lugar de gastar el tiempo en la construcción de infraestructura.
